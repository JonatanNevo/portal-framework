## Platform Support

| Platform | Compiler               |
|----------|------------------------|
| Windows  | MSVC 2022+ / Clang 20+ |
| Linux    | GCC 14+ / Clang 20+    |
| macOS    | Clang 20+              |

---

## Requirements

- **C++23** compatible compiler
- **CMake** 3.25+
- **Vulkan SDK** 1.4+
- **vcpkg** (for dependencies)

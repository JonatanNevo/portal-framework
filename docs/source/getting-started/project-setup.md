# Project Setup

## Using Portal Tool 

If you don't have `portal-tool` insalled, please refer to [Installation](../portal-tool/installation.md)  

`portal-tool` has an interactive wizard to help you setup your project.  
Simply run:
```shell
portal-tool init
```

This will ask you the following questions:

#### Project Name
This will be used as the folder name

#### Vcpkg Installation Method
Either local (submodule) or global if installed

#### Example to use
If you want a blank project or use an example

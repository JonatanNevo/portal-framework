## Quick Reference

### Branch Naming

```text
feature/description    # New features
fix/description        # Bug fixes
docs/description       # Documentation
refactor/description   # Code refactoring
```

### Commit Messages

```text
(module): Short description

Longer explanation if needed.

Fixes #123
```

# Tutorials

tep-by-step guides to learn Portal Engine from the ground up.

Each tutorial builds on the previous one. Start with the first and work your way through.

---

## Learning Path

| # | Tutorial                                               | Time   | What You'll Learn                                 |
|---|--------------------------------------------------------|--------|---------------------------------------------------|
| 1 | [Window and Input](01-window-and-inputs.md)            | 15 min | Application setup, window creation, input polling |
| 2 | [Rendering Basics](02-rendering-basics.md)             | 30 min | Renderer initialization, clearing, draw commands  |
| 3 | [Scene Graph](03-scene-graph.md)                       | 30 min | Nodes, transforms, parent-child relationships     |
| 4 | [Materials and Shaders](04-materials-and-shaders.md)   | 45 min | Shader loading, materials, pipelines              |
| 5 | [Building a Simple Game](05-building-a-simple-game.md) | 60 min | Putting it all together                           |

## Prerequisites

Before starting, make sure you have:

- Completed the [Installation](../getting-started/installation.md)
- Run the [Quickstart](../getting-started/quickstart.md) successfully
- Basic C++ knowledge (classes, templates, lambdas)
